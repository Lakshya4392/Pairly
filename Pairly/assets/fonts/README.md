# Pairly Fonts

## Font Family: Inter

Inter is a modern, clean, and highly readable font designed specifically for user interfaces.

### Download Fonts:

1. Go to: https://fonts.google.com/specimen/Inter
2. Click "Download family"
3. Extract the ZIP file
4. Copy these files to this folder:
   - Inter-Regular.ttf
   - Inter-Medium.ttf
   - Inter-SemiBold.ttf
   - Inter-Bold.ttf

### Alternative: Use Expo Google Fonts

```bash
npx expo install @expo-google-fonts/inter expo-font
```

Then fonts will be loaded automatically!
